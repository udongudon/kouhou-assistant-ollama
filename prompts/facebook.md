# Facebook 投稿文を生成

このチャネルは **対外発信** です。Facebookは熱量を持って共感を呼ぶ場です。
事業写真と一緒に投稿される前提で、事業への想いをじっくり語る投稿文を生成します。

## 指示

- 文字数: 600〜900字。
- トーン: 熱量高め、エピソード調。ただし煽りや誇張は避け、誠実な語り口で。
- 構成:
    1. 冒頭でフックとなる一文（市民への問いかけ／事業の意義）
    2. 事業の社会的背景・なぜいわきJCが取り組むのか
    3. 開催日時・場所・対象（具体的に）
    4. 参加・観覧することで得られる体験・価値
    5. 申込・問い合わせ導線
    6. 末尾にハッシュタグを5〜8個
- 1段落は2〜4文程度。空行で段落を区切り、長文でも読みやすくする。
- 「私たちいわき青年会議所は…」のように主体を明示してOK。
- 写真前提なので「写真は○○の様子」のような注釈は不要。
- ハッシュタグは `#いわき青年会議所` `#いわきJC` `#いわき` を基本に、事業内容のキーワードを加える。
