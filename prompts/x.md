# X（旧Twitter）向け 投稿セットを生成

このチャネルは **対外発信** です。一般の方に興味を持ってもらうのが目的。
1議案から **5本の投稿** を時系列でセット生成します。

## 指示

- 各投稿は本文140字以内（URL・ハッシュタグは別枠扱いの想定）。
- トーン: 短く、テンポよく、興味喚起。煽りすぎない。
- 各投稿で改行を活用し、スマホで読みやすくする。
- 絵文字は1〜2個まで。
- ハッシュタグは各投稿の末尾に2〜4個。`#いわき青年会議所` を基本入れる。

## 出力フォーマット（厳守）

各投稿は以下のフォーマットで連続して出力してください。コードブロックは使わない。
必ず5本すべて出力してください。1本だけ、または一部の投稿だけで終わらせないでください。
見出しは `【告知】` `【1週間前】` `【前日】` `【当日】` `【事後報告】` の完全一致にしてください。

```
【告知】
（本文）

【1週間前】
（本文）

【前日】
（本文）

【当日】
（本文）

【事後報告】
（本文）
```

## 各投稿の役割

- **告知**: 事業の意義 + 開催情報の核 + 申込URL
- **1週間前**: 申込締切のリマインド + 当日の見どころ
- **前日**: 明日開催のリマインド + 持ち物・天候・会場アクセス等の補足
- **当日**: 「これから始まります」感。会場の雰囲気を伝える前提（写真同時投稿想定）
- **事後報告**: 開催のお礼 + 印象的だった瞬間の言語化 + 次回・関連事業への接続

事後報告は議案書時点で詳細不明なため、テンプレ的に「○○な熱気のなか無事に開催できました」程度の汎用文でOKです。差し替えやすいよう、固有名詞や数字に頼りすぎない文にしてください。
