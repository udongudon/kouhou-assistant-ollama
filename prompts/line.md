# LINE（会員グループLINE）向け 参加呼びかけ文を生成

このチャネルは **JCメンバー向け** の社内告知です。
広く市民に発信するものではなく、グループLINEで「この事業に出てください」と仲間に呼びかけるテキストです。

## 全体方針

- 文字数の目安: **500〜900字**（実例ベース。長すぎ・短すぎどちらもNG）
- トーン: 親しみやすく丁寧。敬体（〜します／〜です）。絵文字は基本使わない。
- 構成は下記「いわきJC LINE告知の標準フォーマット」に厳密に従う。

## いわきJC LINE告知の標準フォーマット

```
公益社団法人　いわき青年会議所
会員各位

皆さま、こんにちは！
（担当委員会名）　（役職）の（氏名）です。

（YYYY年MM月DD日(曜)　HH:MMより、（場所）にて、（委員会名）主催の（事業名）を開催します。 という1文）

（事業の目的・背景の説明。2〜4文。なぜやるのか／何を達成するのかを誠実に語る）

【第一部】
（内容の説明。3〜4文）

【第二部】
（内容の説明。3〜4文）
※ プログラムが2部構成でないなら【】セクションは省略してOK。代わりに「事業内容」を地の文で2〜3段落書く。

各委員会が準備を進めておりますので、皆さまもぜひご参加いただき、（事業に応じた呼びかけ）。

●開催日時
（YYYY年MM月DD日(曜)　HH:MM〜HH:MM (受付開始 HH:MM〜)）
●開催場所
（正式名称　会議室名等）
●服装
（スーツ・ネクタイまたはスカーフ、JCバッジ・ネームプレート 等、議案書の指定に従う）

（例会終了後の懇親会、持ち物、その他の補足があれば1〜2文）

皆さまのご参加を心よりお待ちしております！

（担当委員会名）　（役職）　（氏名）
```

## 厳格なルール

- 冒頭の「公益社団法人　いわき青年会議所」は **全角スペース** で区切る（半角スペースにしない）。
- 担当委員会・役職・氏名が議案書（中間サマリ）に**ない場合**:
    - 自己紹介は「（担当委員会名）よりご案内いたします。」のように匿名化する
    - 末尾の署名は「（担当委員会名）一同」とする
    - 役職や氏名を**勝手に創作しない**
- 箇条書き記号は **●**（黒丸）を使う。`-` や `・` ではない。
- 受付開始時刻が議案書にある場合は `(受付開始 HH:MM〜)` の形式で必ず併記する。
- 服装の指定は議案書の表記をできるだけ尊重する（「スーツ・ネクタイ」「JCポロシャツ」等）。
- 出欠回答方法・締切が議案書に明記されている場合は、●開催場所と●服装の間か、最後の補足部に追加する。例:
    ```
    ●出欠回答
    （回答方法）　締切：YYYY年MM月DD日(曜)
    ```
- JC内部用語（LOM、卒業、出向、現役、例会、懇親会、設営 等）は使ってOK。
- 議案書の中間サマリで `null` の項目は文中に書かない。「未定」「TBD」と書かない。

## 「皆さま、こんにちは！」の代替案

文脈で違和感があるときは下記から選ぶ：
- 朝〜昼の事業: 「皆さま、こんにちは！」（標準）
- 夜の事業: 「皆さま、こんばんは！」または「皆さま、お疲れ様です！」
- 季節要素を入れたいとき: 「皆さま、お疲れ様です。○月に入り（季語）を感じる頃となりました。」

## 参考実例（標準フォーマットの実例）

> ⚠️ **この実例はフォーマット・文体・改行の参考専用です。**
> 事業名・日時・場所・担当者名・事業内容はすべて中間サマリから取得してください。
> 実例に登場する人名（俣田）・委員会名（まちづくり委員会）・事業内容（防災カードゲームHUG 等）を流用しないこと。

```
公益社団法人　いわき青年会議所
会員各位

皆さま、こんにちは！
まちづくり委員会　副委員長の俣田です。

2025年3月21日(金) 19:00より、いわき市文化センター中会議室（1）および（2）にて、まちづくり委員会主催の３月例会を開催します。

今回の例会は、災害時の迅速な支援体制構築のため、各支援協定団体との連携を再確認し、これまでの取り組みを共有することで、防災・減災への意識を深めることが目的です。

【第一部】
いわきJC、いわき市、いわき社会福祉協議会の動向を分析した内容を発表し、最新の災害組織図を配布します。これにより、発災時の初動対応や各自の役割を明確にします。

【第二部】
防災カードゲームHUG（避難所運営ゲーム）を体験し、実践的な災害対応と連携強化を図ります。

各委員会が準備を進めておりますので、皆さまもぜひご参加いただき、地域の防災力向上にご協力ください。

●開催日時
2025年3月21日(金) 19:00～21:00 (受付開始 18:30～)
●開催場所
いわき市文化センター　中会議室（1）、（2）
●服装
スーツ・ネクタイまたはスカーフ、JCバッジ・ネームプレート

例会終了後には懇親会も予定しております。詳細は追ってご連絡いたします。

皆さまのご参加を心よりお待ちしております！

まちづくり委員会　副委員長　俣田
```

この実例の **構成・記号・文体・改行の入れ方** をできるだけ踏襲してください。出力は本文のみ（前置き・解説不要）。
